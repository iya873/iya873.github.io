# iya873.github.io
Mod 1 Project Jeopardy Style Music Trivia

<h1>Premise of Game</h1>
<p> The goal of the game is to keep going answering as many questions as you can before getting any wrong. If you get any answers wrong you lose the game. The team side will turn green when it is that team's turn to play. If you get the answer correct, it is the next team's turn and you keep going until someone loses.
 <h2>Languages</h2>
 <p>This game is primarily made up out of HTML,CSS and JavaScript.</p>
